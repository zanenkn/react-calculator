# TDD React

A simple Calculator app created as a practice after a tutorial at [testdriven.io](https://testdriven.io/blog/tdd-with-react-jest-and-enzyme-part-one/)

## Local Setup

```
$ git clone https://github.com/zanenkn/react-calculator.git
```

```
$ cd react-calculator
```

```
$ npm install
```

## Run Locally

```
$ npm start
```

## Run Tests

```
$ npm test
```
